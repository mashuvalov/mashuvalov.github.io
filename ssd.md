# Установка SSD

Установка быстрого SSD накопителя вместо жесткого диска позволяет ускорить работу ноутбука в несколько раз!

Есть 2 варианта апгрейда ноутбука с помощью SSD:

- Установка SSD накопителя вместо жесткого диска.
- Установка SSD накопителя вместо оптического привода (жесткий диск остается на месте).
